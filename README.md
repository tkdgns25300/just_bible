# Just Bible

성경 본문을 빠르게 검색하고 복사할 수 있는 웹 애플리케이션

## 핵심 기능

- 성경 본문 검색 (책, 장, 절 단위)
- 본문 복사 기능
- 반응형 UI

## 기술 스택

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Deployment**: Vercel

## 브랜치 전략

| 브랜치 | 용도 |
|--------|------|
| `main` | 배포용 (항상 안정) |
| `dev` | 개발 통합 |
| `feature/*` | 기능별 브랜치 (예: feature/search, feature/copy) |
